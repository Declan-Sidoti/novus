Introduction:

Novus is a multi-rotor hexacopter unmanned arial system (UAS) with the ability to collect thermal/visible imagery and telemetry data from its on-board computer. The data collected from the unmanned arial vehicle (UAV) is streamed into virtual reality (VR) allowing data visualization and flight control in real time.

LSD SLAM

LSD-SLAM uses the drone’s monocular camera to create point clouds out of its imagery data. This allows us to create a 3D virtual representation of the world using images captured from the drones cameras.

Virtual Reality
Flight Deck

The VR flight deck provides a live on-board video stream display, a virtual map table with the drone’s position overlaid on it, as well as a data visualization framework that keeps critical information immediately available to the operator.
